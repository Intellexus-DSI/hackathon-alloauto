# / /
rnam sras rtogs pa bzhugs /
[ ]@## / /
dpal phyag na rdo rje la phyag 'tshal lo / /
de nas gsang ba bshad par bya / /
zla ba yar ngo'i bco lnga la / /
dam tshig ldan pa'i rnal 'byor bas / /
thun mong ma yin mchod pa ste / /
gang gis bdugs pa tsam gyis ni / /
dngos grub myur du thob byed pa'i / /
sgrub pos bdug spos bdug pa ni / /
brtul zhugs 'dzin pas 'di ltar bya / /
'bras dkar btags pa'i phye ma ni / /
sardza ra dza dag dang ni / /
ga pur tsan+dan dkar dmar dang / /
cha mnyam nas ni de dag bsre / /
a ka ru nag gis bsgos nas / /
be ta dkar mo'i char rgyun gyis / /
rnam par sbrus nas rab bsngags te / /
rtsa ba'i rig pa gang de la / /
dbab pa'i gsang sngags rab tu sbyar / /
stong rtsa brgyad du bzlas nas ni / /
dngos grub phyir ni rdzas de yi / /
de nas mkhas pas ri lu bya / /
gang zhig mchod pa'i dus su ni / /
sems kyi dbang phyug ting 'dzin gyis / /
rnam par sbyangs pa'i sa gzhi la / /
rten sogs dgram par byas nas ni / /
chogs gter bdag mdun bkug nas / /
rdo rje can gyis bdug pa dbul / /
gser sogs mdzes pa'i snod du ni / /
tsan+dan la sogs mdag me la / /
sngar bshad pa yi ri lu bsreg /
des ni rten la bdug par bya / /
sbyor nges skyob pa mnyes gyur na / /
gter rnams de yi mod la ster / /
gal te de ltar ma gyur na / /
gter bdag nyams par gdon mi za / /
'di ni rab tu gsang ba ste / /
gsang ba'i gnas su spyad par bya / /
bdug spos kyi sbyor ba'o / /
de nas dam tshig bshad bya ba / /
rtag par khrus dang gtsang sbra bya / /
rnga dril la sogs rol mo'i sgra / /
sna tshogs me tog dri ldan dang / /
sna tshogs 'bru yi chan dang bshos / /
mdzes pa'i dar snas rtag mchod bya / /
chang rnams dbul bar mi bya ste / /
sha ni brtags te dbul bar bya / /
rtag tu sngon po'i gos 'chang zhing / /
lung thang 'bras bu'i bgrang 'phreng bgrang / /
myur 'gro sha yi bza' ba spang / /
nyams shing chag pa'i snod mi bza' / /
rtsub cing nag po'i stan mi 'dug /
bza' bral mas bstabs bza' btung rnams / /
rnam pa kun du mi spyod do / /
srog bral ba la reg pa spang / /
bsrung bar bya ba'i dam tshig go / /
de nas rten ni bshad par bya / /
phyi nang gsang ba'i bye brag gi / /
gter bdag brten pa'i rten yang ni / /
rnam pa gsum du mkhas pas brtag /
phyi rten rta mchog rab sngo la / /
mdzes pa'i rgyan btags mgon la dmigs / /
rtag tu yang ni bcang bar bya / /
yang na gzugs legs rab tu gzung / /
gsang rten gro ga'i shun sogs la / /
gur gum dri bzangs rab spel bas / /
bai yig bris la sngags kyis bskor / /
rin chen dang bcas rin chen gzugs / /
'bru mchog btags nas bsnos byas la / /
mkhas pas byas te snying po gzhug /
tshon byugs nas ni mdun gzhag go / /
gnyis pa bram ze'i ras dag gam / /
gzhon nu ma yis legs bkal ba'i / /
ras la dri yis rab bsgos nas / /
gter bdag sku mdog phyag char ldan / /
lugs nas 'byung ba'i rim pa yis / /
dam tshig ldan pas myur bris la / /
phyi rol shing sogs rgyan gyis mdzes / /
sngags rig pas ni mdun dgram bya / /
rten gzugs pa yi rim pa'o / /
de nas gsang ba bshad bya ste / /
sgrub po gsang bdag phyogs bltas te / /
gtor ma'ang de nyid phyogs su sbyin / /
gsang rten ne'u le'i rlud bur gzhug /
sngon dmar kha spro rgyun 'chang zhing / /
'bras sogs 'bru rnams der gzhug bya / /
gzhan gyis mthong na dngos grub nyams / /
gtor sgrub pa la gsang ba shis / /
cho ga'i sbyor ba 'di yis ni / /
nam mkha' mdzod la rnam par rol / /
rgyal po chen po rnam thos sras kyi rtogs pa bde byed cho ga sna tshogs pa'i mdzod ces bya ba rdzogs so / /
bgegs kyi rgyal po'i sgrub thabs bzhugs /
