gang la gnyer ma lnga yod cing / /
de yi sna rgyud mtho ba dang / /
dpral ba'i mtha' gnyis rgyas shing mtho / /
lus po shin tu sbom la snyoms / /
mgo bo zlum zhing rab tu rgyas / /
sma ra srab cing snyoms pa dang / /
na chung lta bur nu ma rgyas / /
dkar dngas so ni dam zhing snyoms / /
mig gi dbang po rgyas lags la / /
rna ba mdangs snyug chung zhing snyoms / /
phrag gong rab tu rgyas pa dang / /
rkang pa'i sor mo bcu bo ni / /
sha rgyas bkrag bzang bo la gong mtho / /
gang gi mchan khung spu med pa / /
lag pa chung shing sha rgyas pa / /
de rnams mtshan ldan lha yi rigs / /
skom gdung che zhing chung ngu za / /
sha yi zas la rnam par rtsi / /
nad med de yi lus la ni / /
nyi shu rtsa lnga'i lo lon tshe / /
chu yi gnod pa 'byung bar 'gyur / /
yul rnams bskor zhing rnam par 'khyams / /
kun du bde skyid ldan par byed / /
brdzun tshig spangs shing thams cad ster / /
'dod pa chung zhing mi rku ba / /
don rnams smra ba bzhin du sgrub / /
gang zhig mi yi rigs gyur pa / /
gdong ring gyen du gnyer ma can / /
smra ra nyung zhing rna shal ring / /
bsdus pa lag pa rnam par 'gul / /
'bru tshugs su ni rnam par lta / /
lus po rdud cing skems pa dang / /
ba spu mang zhing ring ba dang / /
de yi mig ni gting du song / /
sna ni chung zad lab pa dang / /
lus bong ring zhing brnab sems che / /
mang du za zhing kun la 'khon / /
'dod pa che la gcig tu brjod / /
chos min spyod cing thob tshad za / /
sems ngan de ni mi yi rigs / /
rkang pa ring zhing gdong pa skams / /
rkang lag ser ka mang ba dang / /
gnag cing ba spu med pa dang / /
zas ngan bza' zhing bzang po 'dor / /
mtshan ngan de ni nyi ma 'dod / /
lag pa'i nor rnams nyams par byed / /
de rnams mtshan ngan yi dags mi / /
'gro zhing 'gro zhing sdod pa dang / /
'gro zhing phyogs rnams kun du blta / /
rkang pa sha med rtsa yis gang / /
sma ra che'am yang na med / /
de rnams mtshan ldan dud 'gro'i rigs / /
de rnams rnam par bshad bya ba / /
gang gi dpral ba'i skra mtshams la / /
thig le'am ri mo mngon can chags / /
blo las ldan zhing sdug bsngal thob / /
sna yi g.yas g.yon mkhur ba la /
sha mtshan nag po chags gyur na / /
zla drug lon nas nad kyi 'debs / /
mig la nag po chags pa na / /
nor yod zas kyis phongs par 'gyur / /
gdong pa gtsang zhing mdangs bzang na / /
nad med gtsang zhing bde bar 'tsho /
mkhur ba g.yas sam g.yon pa la / /
mdzer pa chung ngu chags pa na / /
longs spyod ldan zhing rnyed pa thob / /
sna 'debs nang na sme ba chags / /
gang zhig sum cu lon pa'i tshe / /
gser dang rta yi rnyed pa ni / /
rgyal po las ni thob par 'gyur / /
mi dpyad 'di rdo rje gdan du u rgyan pas bsgyur pa'o //
