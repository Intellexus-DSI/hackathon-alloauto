bcom ldan 'das kye'i rdo rje la phyag 'tshal lo / /
kye'i rdo rje bdag med ldan / /
mkha' 'gro brgyad kyis yongs skor la / /
phyag 'tshal nas ni phyag mtshan rnams / /
bsnams pa'i phan yon rim bzhin brjod / /
glang po la sogs bsnams pa ni / /
nad rnams zhi bar mdzad pa ste / /
glo dang dbugs dang de bzhin smyo / /
khrag skyug mdze nad 'brum bu dang / /
mcher pa mchin pa'i rang bzhin no / /
glang pa la sogs rim zhes grags / /
sa la sogs pa brgyad rnams pa / /
phyug po thams cad 'grub pa'o / /
nga rgyal la sogs skyon drug po / /
gcod phyir gri gug g.yas pas te / /
dngos dang dngos med rtog pa la / /
g.yon pa na ni pad snod do / /
'khor ba'i rgya mtsho'i dug chu las / /
nya 'dra'i 'gro ba 'don phyir dang / /
bkra shis 'dzin pa ro he ta / /
nyon mongs rgya 'dra gri gug g.yas / /
gti mug gis ni rab rmongs la / /
phag dang 'dra ba'i sems can rnams / /
gti mug gnyid las bslang ba'i phyir / /
g.yas na cang te'u'i sgra sgrogs pa / /
'dod chags kyis gdungs rus sbal ni / /
'gro ba'i nad rnams bsal phyir dang / /
chags chen bdud rtsis tshim bya'i phyir / /
rus sbal dang ni pad snod bsnams / /
zhe sdang dgra yis pham byas pa / /
sbrul dang 'dra ba'i sems can rnams / /
zhe sdang chen pos tshim bya'i phyir / /
phyag na sbrul dang thod pa'o / /
ngan rtog pa yi ri dgas rnams / /
skrag par bya phyir seng ge'am / /
seng 'dra nga rgyal gzhom phyir ram / /
seng ge dang ni sgra sta bsnams / /
bdag tu lta ba sogs pa yi / /
lta ba po yi gnyis pa'o / /
bdag tu lta ba'i mdud pa dang / /
ser sna dgrol bar bya ba'i phyir / /
dge slong dang ni 'khor gsil lo / /
dper na thod pa lta bu yi / /
chos ni 'khor lo bskor ba yis / /
nyon mongs tha ba rlog pa'i phyir / /
'khor lo dang ni thod pa bsnams / /
rnam par rtog pa'i ri [(]brnyil [,]bsnyil [)]zhing / /
gdug pa tshar gcad bya ba'i phyir / /
g.yas pa sdigs mdzub bstan pa'o / /
bcom ldan dpal chen he ru ka /
ma mo'i tshogs kyis yongs bskor ba'i / /
phyag mtshan bsnams pa'i yon tan rnams / /
ci nus brjod thob yon tan rnams / /
thabs dang shes rab gru chen dang / /
brten nas mtha' yas sems can rnams / /
'khor ba'i rgya mtsho las brgal te / /
thar pa'i skam sar gnas par shog / /
