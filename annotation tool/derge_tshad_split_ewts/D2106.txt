gdams ngag nye bar bsdus pa bzhugs // @## / /
'phags pa 'jam dpal ye shes sems dpa' la phyag 'tshal lo / /
chos kun rang bzhin 'jam dpal ye shes sems / /
ma lus bde gshegs kun gyi mtshan gyur pa / /
rdo rje 'chang chen rnams kyi glur blangs don / /
zab cing rgya che'i skye bo blo bzangs don / /
byang chub sems ldan lhag pa'i lhar mos la / /
sgyu ma'i rten 'brel rnam gsum sbyor ba las / /
ma lus gnyis med mkha' khyab sprul tshogs dgu'o / /
ma skyes mi 'gag sgyu 'phrul dra ba'i tshogs / /
rdo rje mig gis snang thob bar du blta / /
sgyu ma brag nam mkha'i rang bzhin du / /
yongs su shes nas ma zhen sems kyis bklag /
gsang sngags tshul chen lam ni 'di yin te / /
ji ltar dngos rnams mkha' la gnas pa bzhin / /
dam tshig dkyil 'khor phyag rgya che la gnas / /
don de mi shes dka' ba'i lam gzhan gyis / /
'khor ba'i grong gsum dag la de mi g.yo / /
de lta'i don shes rang bzhin nyes pa spangs / /
spos kyi glang po ke ta ka zos bzhin / /
nyes pa'i tshogs bcom gzhung tshogs yon tan ldan / /
'gag pa med pa'i chos kyi rgyal por 'gyur / /
thugs kyi gsang mdzod gtsug gi rin po che / /
'gro kun sems kyi rang bzhin chos rnams lung / /
rgyud kun dbang po rdo rje tshig tu gnas / /
mkhan po'i rjes 'brang nye bar gyur ba yin / /
slob dpon chen po dpal mai tri'i gdams ngag nye bar bsdus pa rdzogs so //
