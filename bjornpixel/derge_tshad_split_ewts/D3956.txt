rang gi bya ba'i rim pa bskul ba dang bcas pa yi ger bris pa /
sangs rgyas dang byang chub sems dpa' thams cad la phyag 'tshal lo / /
'khor ba thog ma med pa nas / /
nyon mongs bag chags tshad med pas / /
yul ngan rkyen dang phrad pa las / /
nyon mongs sdug bsngal skye ba'i tshe / /
rang gi dran pa'i phyir du 'di / /
byas pas de tshe dran par shog /
nang gi sems ni dul gyur na / /
phyi yi dgra 'dis gnod mi nus / /
rang gi nang sems g.yo ba na / /
phyi yi dgra 'dis rkyen byas nas / /
nang gi dgra 'dis rang rgyud bsreg /
des na nang gi dgra 'di gzhom / /
dug can snod kyis zas brlag bzhin / /
nang gi dri mas dge tshogs brlag /
gnyen po chu yis khrus byas nas / /
dag pas bde ba chen po 'thob / /
gang na gnas tshe gang gi tshe / /
gang gi dri ma ldang ba dang / /
nang gi dgra 'di 'byung ba'i tshe / /
thabs ni sna tshogs gsungs pas spang / /
dgra gzhom dri ma bkru ba'i phyir / /
de tshe 'bad pa chen po bya / /
gang gis rang gi rgyud 'di la / /
nang du bltas te rang sems brtag /
de tshe yul la shes pa skye / /
shes pa skye ba'i yul rnams gsum / /
nyon mongs bar ma dge ba skye / /
rgyu rkyen yul gyi bye brag gis / /
nyon mongs skye ba dag la gsum / /
chen po 'bring dang chung ngu'o / /
gnyen po'i rim pa'ang de bzhin te / /
chen po skye ba'i yul de la / /
brten te tshogs chen 'bad pas skye / /
'bring gi yul la 'bring yin te / /
chung ngu la ni chung ngu'o / /
chen po kun la skyes na mchog /
bar ma lung ma bstan pa yang / /
dran pas zin pa'i sems de yis / /
kun slong rjes kyis dge bar bsgyur / /
thabs sgo sna tshogs pa yis so / /
dge ba che 'bring chung ngu dag /
ngang gis skyes pa'i yul de la / /
sbyor ba dngos dang rjes dag las / /
khyad 'phags bsod nams kun bsngo bya / /
shes na gang la'ang tshogs 'phel 'gyur / /
thog med dus nas ji srid du / /
bdag ni dug gsum zhen pas 'khor / /
ji srid skye shi kha brgyud pas / /
de srid las rnams chud re zos / /
ji srid dge sems 'phel 'gyur ba / /
de srid sangs rgyas mngon sangs rgyas / /
bdag ni gti mug mun pas bsgribs / /
mun nag glog 'gyu snang ba tsam / /
ci zhig ltar stes dran rnyed tshe / /
de tshe 'bad pas bsgrub par bya / /
'chi bar nges pa cha yod cing / /
gnas pa'i dbang ster med mod kyi / /
bdag ni re zhig mi 'chi snyam /
dran pa btang ste byang chub bsgrub / /
gzhan dag la yang sgrub par gyis / /
dper na byis pa rig med pas / /
me dang g.yang sogs 'jigs pa med / /
de bzhin 'gro ba gti mug rnams / /
ngan song dag la'ang 'jigs pa med / /
dper na zas mchog ro dang ldan / /
nad pa dag la byin pa'i tshe / /
zas la smod cing mi 'dod bzhin / /
thabs kyi bye brag 'di dag kyang / /
rmongs pa nyon mongs ldan pa la / /
bstan kyang de 'dra nyid du'o / /
dper na bza' la dga' ba'i mi / /
zos brlag bsags phan cha yod kyang / /
'dod pas mi tshugs za ba bzhin / /
snga phyi rgyu 'bras cha yod kyang / /
nyon mongs zhen pa stobs cha yod / /
dge med sdig pa sna tshogs spyod / /
dper na kha bya mkhan dag gis / /
kha la bya dgos kun bsgrub zer / /
gzhan gyi dran pa bskyed pa'i tshe / /
de bzhin dge ba bya dgos zer / /
kha cig kha la gleng byed cing / /
zhe la don du sgrub mi byed / /
kha cig 'bras bu don chen po / /
btang nas bya ba chung ngu sgrub / /
de yang byis pa'i spyod pa can / /
kha cig bsam pa rlung bzhin g.yo / /
rung dang mi rung gnyis ka byed / /
kha cig kha dog dkar po bzhin / /
rkyen gang ldan pa de ltar 'gyur / /
kha cig rang nyid ngang gis dge /
grogs kyis rbad pas nyon mongs skyed / /
kha cig rkyen med rgyu dang ldan / /
dam pa'i grogs kyis zin pa na / /
sdig spong dge byed de la bskul / /
kha cig dge la bskul mi dgos / /
